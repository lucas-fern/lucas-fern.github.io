# Run Project

### 1. Clone the project

### 2. Run the project

```shell
npm i
npm start
```

### 3. Build

```shell
npm run build
```

# Updating

- [ ] Update content in `public` on `main` branch
- [ ] Update `meta description` in the head of `public/index.html`
- [ ] `npm run build` and move the contents of `build` to the `gh-pages` branch
